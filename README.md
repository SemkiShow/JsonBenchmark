# Json Benchmark

A test project for benchmarking different json libraries

## How to run

- Download the latest release
- Unzip it
- Run JsonBenchmark or JsonBenchmark.exe depending on your system

## How to build

1. Run 
```bash
git clone https://github.com/SemkiShow/JsonBenchmark.git --recurse-submodules
cd JsonBenchmark
chmod +x run.sh
./run.sh
```
