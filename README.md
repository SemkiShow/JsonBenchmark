# Json Benchmark

A test project for benchmarking different json libraries

## How to run

- Download the latest release
- Unzip it
- Run JsonBenchmark or JsonBenchmark.exe depending on your system

## How to build

1. Run 
```bash
git clone https://github.com/SemkiShow/JsonBenchmark
cd JsonBenchmark
git submodule update --init --recursive --depth 1 --jobs 8
chmod +x run.sh
./run.sh
```
